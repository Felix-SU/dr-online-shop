# Online shop
Digtal Resources online shop using PHP and Ajax
<br>
<b>Requiment</b>
<ul>
  <li>PHP  7.3.0</li>
  <li>Apache|Nginx</li>
  <li>MySQL|MariaDB</li>
  <li>Better use 1920x1080 Resolution</li>
</ul>
<br>
Install & Run
<ul>
  <li>import db.sql</li>
  <li>Configure database connection in db/conn.php</li>
  <li>And email in php/send.php and php/reg.php</li>
</ul>
